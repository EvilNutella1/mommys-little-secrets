Here is a list of Elements you can use to format and style your admin texts.
Only focus on the Elements in the beginning and ending of example! For example: <x> Text </x> | The Elements are <x> and </x>.
Everything between the Elements is going to be your own text of choice.
Tipp: Copy paste the examples and just replace the inside and fill it with whatever text you want.

  1. <b> bolt </b>

  2. <i> italic/cursiv </i>

  3. <h1>

    Headline

    </h1>

  4. <u> underline </u>

    5. If you want to combinate 2 Elements then always start with h1 and then add the other Element inside:

  5. <h1> <u>

    Headline + underline

    </u> </h1>
